# portal-official-notes

本仓库用于归档和发布多个独立 HTML 页面，主要存放各类说明、记录与信息展示页面。仓库本身不绑定任何特定域名或网站，仅作为静态页面的集中存放与管理空间。

## 项目简介

portal-official-notes 是一个通用型 HTML 页面归档仓库。所有页面均为独立、自包含的 HTML 文件，适用于信息展示、通知发布、文档说明等场景。仓库结构简单，便于快速浏览与维护。

## 目录说明

```
portal-official-notes/
├── pages/           # 存放所有独立 HTML 页面
├── assets/          # 公共资源文件（样式、脚本、图片等）
├── index.html       # 仓库首页 / 页面导航
└── README.md        # 本文件
```

- **pages/**：各 HTML 页面按功能或主题分类存放，文件名清晰标识页面内容。
- **assets/**：存放多个页面共享的 CSS、JavaScript、图片等资源，避免重复引用。
- **index.html**：提供页面索引或列表，方便快速定位与访问。

## 页面归档说明

- 所有归档页面均为纯 HTML 格式，不依赖外部服务器或数据库。
- 页面内容以信息记录与说明为主，不包含商业推广、诱导点击或具体营销承诺。
- 页面文件名与标题保持对应，便于识别。
- 如需新增页面，请放入 `pages/` 目录，并在 `index.html` 中添加导航链接。

## 维护说明

- 本仓库由维护者定期更新，确保页面内容准确、链接有效。
- 若发现页面错误或内容过时，欢迎提交 Issue 或 Pull Request。
- 仓库不承担任何因页面内容使用而产生的责任，所有页面仅供参考或记录用途。
- 维护者保留对页面内容进行修改、删除或归档的权利，恕不另行通知。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的对应页面。
3. 也可通过 GitHub Pages 等服务进行在线访问（需自行配置）。

## 贡献指南

- 提交新页面时，请确保 HTML 结构完整、无外部恶意链接。
- 页面样式尽量简洁，避免过度依赖外部资源。
- 如需修改已有页面，请在 Pull Request 中说明修改原因。
